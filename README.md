> Un pipeline MLOps complet : de l'entraînement d'un modèle Random Forest sur le dataset **Mobile Price Classification** (Kaggle) jusqu'au déploiement d'une interface interactive via Docker.

---

## 🎯 Objectifs du Projet
* **Entraînement automatisé** : Script robuste avec évaluation des performances.
* **Interface Interactive** : UI Streamlit pour tester le modèle en temps réel.
* **Conteneurisation** : Image Docker optimisée pour un déploiement "Zero Config".
* **CI/CD Pipeline** : Automatisation complète via GitHub Actions (Train > Build > Push).

---

## 🛠️ Stack Technique
* **Data** : Scikit-learn, Pandas, Numpy.
* **Interface** : Streamlit (Web UI).
* **DevOps** : Docker, Docker Hub.
* **Automatisation** : GitHub Actions.
